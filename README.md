# Material UI Sample

## Setup

```
$ git clone https://github.com/KeitaMoromizato/js-study-material-ui
$ cd js-study-material-ui
$ npm install
$ npm start
```

## Stack

* React
* Webpack/WebpackDevServer
* stylus-loader
* Babel(ES2015, class-property)
